
### nodejs101

A barebones Node.js app using [Express 4](http://expressjs.com/).

## Running Locally


```sh
git clone git@github.com:k8snative/nodejs101.git # or clone your own fork
cd nodejs101
npm install
npm start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).
